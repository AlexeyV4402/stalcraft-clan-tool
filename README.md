# stalcraft-clan-tool
Консольный инструментарий на Rust для администрирования клана.
Реализовано:
Запись участников клана из Stalcraft API в базу данных.
Проверка .xlsx файла отрядов (столбцы - отряды, строки - участники; 1-е столбец и строка использовать игнорируются при провеке, поэтому пишите с (2, 2) клетки). Удаляются дубликаты людей из таблицы, удаляются люди, отсутствующие в БД участников, выводятся люди, отсутствующие в отрядах.
Считывание списка участников со скриншота КВ и последующая запись в "черновик". Верхняя граница скриншота должна быть точно началом таблицы участников. Отступы слева и права регулируются в Config.toml
Запуск с аргументом --path="<путь скриншота>" . --announce, отправляет уведомление об успешности через dbus. Уведомления Gnome работают, остальное не тестил. Также есть готовое расширение Gnome, которое автоматически запускает данную программу через bash. Если нужно это расширение - пишите в issues.
Система черновиков - при загрузке скриншота или при ручном создании, в отдельной базе данных создаётся черновик, в который записываются введённые данные (со скриншота автоматически считываются люди из статистики). Задумывались удобное чтение и вывод информации с черновиков, но это не доделано. По плану, черновики должны были позже записываться как постоянные записи в долгосрочную базу данных.
